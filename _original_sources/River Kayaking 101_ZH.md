傾斜動能 / 側傾平衡 (TILT MOMENTUM)
	低位支撐 (Low Brace)
	高位支撐 (High Brace) - 解釋保持肘部朝下以產生槳葉阻力。如果對側肘部抬起，槳葉會向下潛入水中。


翻滾教學 (ROLL lesson)
示範 C to C 翻滾 (以右手為主，即使是左撇子也建議右手控制槳葉)。
翻滾步驟解析：
第一步 - 設定 (Set up)
	第二步 - 左大拇指靠在臀部/座艙位置，右手向下推向左側。
	第三步 - 拉槳 + 髖部翻正 + 頭部保持朝下，右肘向右膝抬起 + 左肘下壓。

指導員站在水中協助學員調整身體位置。協助學員「感覺」他們的身體位置。因為水下一切都是反向且倒置的，單用認知解釋很難理解。最好是透過體驗來感受正確的身體位置。握住划槳協助學員翻正 3-5 次。

如果成功 - 嘗試獨立完成。
如果不成功 - 保持在第二步，指導員從後方勾住學員右腋下，協助學員從該位置嘗試翻正。

常態化 (NORMALIZE) - 將新奇行為常態化。這樣在嘗試翻滾時就不需要刻意去認知動作細節。
進行髖部翻正時，必須練習保持頭部朝下，以便將此行為常態化，這樣在進行完整翻滾時才能專注於其他步驟。
左肘下壓 - 解釋如果沒做好，槳葉會如何「潛入」水中。
